# LumiSkin Project

## Deskripsi
LumiSkin adalah sistem analisis kondisi wajah dan rekomendasi skincare berbasis AI.

## Tech Stack
- Frontend: React (Vite)
- Backend: Express.js
- AI: Rule-based (sementara)
- Data Science: Streamlit (planned)

## Cara Menjalankan

### Backend
cd backend
node app.js

### Frontend
cd frontend
npm install
npm run dev

## Struktur
- frontend: tampilan UI
- backend: API & logic
- ai-model: model AI
- data-science: analisis data
